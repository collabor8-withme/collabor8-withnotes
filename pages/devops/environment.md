# 环境变量
## 环境变量是什么
他是存在系统内的一个全局变量

## 种类
### 用户变量
专属与某个用户的全局变量

### 系统变量
所有用户都可以查看使用的全局变量


环境变量有很多
- PATH
- HOME


怎么去添加一个环境变量
- 临时添加
``` shell
zhouxin=1234
export zhouxin
```

- 永久添加
```

export PATH

source .bashrc
```
